===== But: =====

gérer les dépendances front-end avec bower.

===== Étapes Pas-à-pas: =====

1. Installer bower (si ce n'est déjà fait) en suivant les instructions dans [[:Installation de poste:bower]]
2. Se placer à la racine du projet, par exemple :

'''
cd pic_unicef/developpement/implementation/symfony
'''

3. Éxecuter la commande suivante :

'''
bower install
'''
