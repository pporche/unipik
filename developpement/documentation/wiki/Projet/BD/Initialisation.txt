===== But: =====

Initialiser une nouvelle base de donnée

===== Étapes Pas-à-pas : =====

1. Dans un terminal, si la base de données est déjà présente  : dans le dossier scriptsBD, exécuter dropdb.sh

2. Dans le dossier scriptsBD, executer ''createdb.sh''  puis  ''addAdmin.sh. ''
	-> createdb.sh demande une fois le mot de passe ulitisateur et deux fois le mot de passe de la base de données

3. Dans le dossier scriptsBD, pour remplir la base de données en entier, executer generateRequestsFile.sh, puis insertAll.sh
