Bonjour,
Si vous êtes ici c'est que vous cherchez à nettoyer les fichiers excel.

1) Placez tout les fichier excels que vous voulez lancer dans le répertoire fichierNettoyer

2) Il faut ABSOLUMENT : *ENLEVER les titres qui apparraissent sur une ligne
dans tout les fichiers. 
*Vous n'avez cependant pas besoin d'enlever les en-tête des colomnes,
ceux-ci sont ignorés lors du traitement.
*PRECISEZ des en-tête bidons pour le fichier de geolocalisation là ou il y a les précisions.

3) *Lancez le script script scriptLancement via la console et ENJOY :) 

Cas spéciaux :* Si vous désirez regénérer seulement un fichier placez vous dans le dossier
scriptNettoyage et lancez le script correspondant au fichier à nettoyer.

*Si vous voulez modifier le processus de traitement d'un fichier spécifique placez-vous dans le fichier traitement et modifier le traitement voulu. 

* Si rien ne passe faites signe à Kafui pour la modification du script.
