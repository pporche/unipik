===== Par où commencer ? =====

Les étapes à suivre pour une personne qui commence à partir de zéro:

1. Si ce n'est déjà fait, installer un OS sur un nouveau poste (ubuntu recommandé), installer git, cloner le dépôt git du projet, installer zim et ouvrir le fichier notebook.zim situé dans le dossier courant.
2. Configurer le proxy en suivant les démarches expliqués dans :Proxy, suivre uniquement les démarches des pages qui correspondent à votre installation, typiquement, [[:Proxy:Linux]] , [[:Proxy:Linux:apt-get]] et [[:Proxy:Firefox]] .
3. Suivre les démarche décrites dans [[:Installation_de_poste]] 
4. Si vous suivez une formation symfony, des informations utiles se trouvent dans [[:Symfony]] 
5. Si vous voulez commencer à travailler sur le projet, des informations utiles se trouvent dans [[:Projet]]
6. Enfin, si vous voulez des informations sur l'administration des serveurs, routeur et autres parefeux utilisé pour le pic ou pour le déploiement du projet, des informations utiles se trouvent respectivement dans [[:Administration_système:PIC]] et [[Administration système:Projet]]
